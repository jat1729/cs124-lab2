Create a New Item in Empty List

Before Adding a Task

You add an element by pressing the "+" button at the bottom of the screen

Selecting a Task

You press the "New Task" button

Naming the New Task

You enter in your new task in the text box labeled "New Task"

Submitting New Task

You press the "Add Task" button to submit the task to the list

After Adding a Task

Task is displayed in the main screen

Create a New Item in Non-Empty List

Before Adding a Task

You add an element by pressing the "+" button at the bottom of the screen

Selecting a Task

You press the "New Task" button

Naming the New Task

You enter in your new task in the text box labeled "New Task".

You also select which folder you want the task to be under

You press the "Add Task" button to submit the task to the list

After Adding a Task

Task is displayed in the main screen

Marking Items Complete

Before Marking Items Complete

You mark items complete by selecting the check mark button on the right side of each task

After Marking Items Complete

Items that are completed are signified with a line through the task

Renaming Items

Before Renaming Items

You rename items by selecting the edit button (notepad) on the right side of each task

After Renaming Items

As seen here, we renamed "Text John" to "Text John about bank statements"

Showing Uncompleted Items

Before Showing Uncompleted Items

You show uncompleted items by selecting the "Tasks to Complete" button on the left side of the task bar

After Showing Uncompleted Items

All the completed items are absent as there are no items with a line through them

Deleting All Completed Items

Before Deleting All Completed Items

You delete completed items by selecting the trash button on the right side of the task bar

After Deleting All Completed Items

All the completed items are deleted as there are no items with a line through them

Adding a new Folder

Before Adding a Folder

You add an element by pressing the "+" button at the bottom of the screen

Selecting a Folder

You press the "New Folder" button

Naming the New Folder

You enter in your new folder name into the text box labeled "New Folder"

Submitting New Folder

You press the "Add Folder" button to submit the folder to the list

After Adding a Folder

The new folder appears in the main screen

Collapsing a Folder

Before Collapsing a Folder

You press the chevron next to the folder you want to collapse

After Collapsing a Folder

Task under the folder clicked disappear
